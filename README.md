# Voiture-control-e-par-le-smartphone
Ce projet consiste à concevoir et programmer une voiture télécommandée basée sur un microcontrôleur ESP32, entièrement développée à l’aide d'Arduino pour la partie programmation embarquée et de HTML/CSS pour l’interface de contrôle. La voiture est équipée de moteurs DC pilotés par deux motor drivers, permettant le contrôle précis des déplacements et de la vitesse.

L’ESP32 crée son propre réseau Wi-Fi, ce qui permet de contrôler la voiture sans application externe, directement depuis un navigateur web sur téléphone, ordinateur ou tablette.
