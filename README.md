# Voiture-control-e-par-le-smartphone
Ce projet consiste à concevoir et programmer une voiture télécommandée basée sur un microcontrôleur ESP32. La voiture est équipée de moteurs DC pilotés par deux motor drivers, permettant le contrôle précis des déplacements et de la vitesse. Le système est commandé à distance via une interface web, accessible depuis un téléphone, un ordinateur ou une tablette, sans avoir besoin d’une connexion Internet.

L’ESP32 crée son propre réseau Wi-Fi, auquel l’utilisateur se connecte directement pour piloter la voiture en temps réel.
